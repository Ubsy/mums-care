Setup Required

This checklist needs to be configured with a free Pantry ID (takes 2 minutes).

  1. Go to getpantry.cloud
  2. Enter your email, click "Create Pantry"
  3. Copy the Pantry ID it gives you
  4. Open this HTML file in a text editor (e.g. Notepad)
  5. Find the line: const PANTRY_ID = '17f5db4d-7978-4bbf-8052-45e20a3b4e60';
  6. Replace 17f5db4d-7978-4bbf-8052-45e20a3b4e60 with your Pantry ID (keep the quotes)
  7. Save the file and re-upload
